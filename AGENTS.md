# AGENTS

Project Cascade is a modern real-time strategy game.

Rules:

- Use ES Modules.
- Use classes.
- One class per file.
- Never create global variables.
- Never put multiple systems into one file.
- Renderer must never contain game logic.
- Game logic must never draw directly.
- Prefer composition over inheritance.
- Keep files small and focused.
- Use English for all code, identifiers, comments, and commit messages.
