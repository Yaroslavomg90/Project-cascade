# Project Cascade Classic
## Game Design Document
### Version 1.0

---

# Цель проекта

Project Cascade Classic — первая полностью играбельная версия Project Cascade.

Главная цель проекта — создать современную RTS, вдохновленную Dune II, Command & Conquer, Red Alert 2, Emperor: Battle for Dune и StarCraft.

Classic должна стать законченной самостоятельной игрой, а не технической демоверсией.

Все механики, не входящие в данный документ, автоматически относятся к Project Cascade Ultimate.

---

# Основные принципы

Classic должна быть:

- простой для освоения;
- глубокой в стратегии;
- динамичной;
- хорошо читаемой визуально;
- легко расширяемой.

---

# Основной игровой цикл

Игрок:

1. Строит базу.
2. Добывает ресурсы.
3. Разведывает карту.
4. Расширяет экономику.
5. Производит армию.
6. Захватывает территорию.
7. Уничтожает противника.

---

# Фракции

В Classic входят три фракции.

## UCA

United Continental Alliance

Особенности:

- тяжелая броня;
- мощная техника;
- высокая живучесть.

---

## EBD

Eurasian Bio Dominion

Особенности:

- биотехнологии;
- агрессивный стиль;
- высокая мобильность.

---

## PQC

Pacific Quantum Collective

Особенности:

- высокие технологии;
- энергетическое вооружение;
- дорогие, но эффективные юниты.

- Легкий броневик

Обязательно входят:

- строительство базы;
- добыча ресурсов;
- энергоснабжение базы;

- туман войны;

- ремонт зданий;

- атака по приказу;

---

# Искусственный интеллект

AI должен уметь:

- строить базу;
- добывать ресурсы;
- расширяться;
- производить армию;
- атаковать;
- защищаться;
- использовать авиацию;
- использовать артиллерию.

---

# Пользовательский интерфейс

Classic использует интерфейс в стиле Command & Conquer.

Обязательно:

- боковая панель справа;
- миникарта;
- список выбранных юнитов;
- очередь строительства;
- очередь производства;
- подсказки;
- иконки способностей.

---

# Управление

Управление максимально близко к StarCraft.

Поддерживаются:

- Drag Select;
- Shift Queue;
- Ctrl Groups;
- двойной клик;
- горячие клавиши;
- правый клик как основная команда.

---

# Графика

Используется:

- изометрический вид сверху;
- современный 2D Canvas;
- спрайтовая графика;
- анимации;
- частицы;
- динамические тени;
- эффекты взрывов;
- дым;
- следы техники.

---

# Звуковое оформление

Каждая фракция должна иметь:

- собственный голос оператора;
- собственные звуки оружия;
- собственные звуки техники;
- уникальные предупреждения.

---

- Пожиратели;
- Катаклизмы;
- Супероружие;
- Биоматериал;
- Уникальная энергия;
- T3 технологии;
- Орбитальные события;
- Кампания;
- Кооператив;
- Сетевая игра;
- Редактор карт;
- Модификации.

---

# Архитектура

Classic разрабатывается как модульная RTS.

Основные принципы:

- ES Modules;
- одна система — один файл;
- Renderer отделен от Game Logic;
- Data Driven подход;
- отсутствие глобальных переменных;
- расширяемость без переписывания существующего кода.

---

# Главная цель версии 1.0

После завершения Project Cascade Classic игрок должен получить полноценную RTS, которая ощущается как современный духовный наследник классических стратегий конца 90-х и начала 2000-х годов.

Project Cascade Ultimate будет развивать Classic, добавляя новые игровые системы без переписывания существующего движка.
